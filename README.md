# ❤️ Heart Disease Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Heart Disease dataset to uncover patterns, trends, and insights using Python.

## 📂 Dataset
The dataset contains patient health information used for heart disease analysis.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Project Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Analysis

## 📈 Key Insights
- Age distribution analysis
- Heart disease target distribution
- Correlation analysis
- Feature relationships

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries.
3. Open the Jupyter Notebook.
4. Run all cells.
